# Programação II - Trabalho Final

O objetivo principal deste trabalho é modelar e implementar um pequeno sistema
orientado a objetos, a partir de um conjunto de requisitos funcionais, aplicando os
conceitos fundamentais de **Programação Orientada a Objetos (POO)**, tais como
*abstração*, *encapsulamento*, *herança*, *polimorfismo* e *relacionamento entre classes*.

O sistema a ser desenvolvido representa o domínio de uma imobiliária, permitindo a
modelagem das entidades envolvidas, seus relacionamentos e comportamentos, bem
como a implementação das principais regras de negócio.
